About this extension

This extension allows you to search the PGP Global Directory, maintained by Symantec (which acquired PGP Corporation in 2010).

OpenPGP standard keys allow an individual to create and send confidential communications to the owner of the public key. In addition, the authenticity of a digitally signed messages can be confirmed using the signer's public key.
